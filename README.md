# Terceiro-Projeto-de-Dados-Programa-Desenvolve-do-Grupo-Boticario

Foi realizado uma análise sobre os  dados da base  de crimes em São Paulo do ano de 2019 e 2020 . O arquivo encontra-se  em anexo . Para uma análise mais detalhada  ocorridos em São Paulo utilizando a base usando o BigQuery para consultas e o Looker para visualização e geração de relatórios. Ao final, você deverá ter aprendido a criar consultas SQL complexas, criar painéis interativos e apresentar insights de dados.
